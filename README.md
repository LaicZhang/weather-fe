## 技术栈

### [前端](https://github.com/LaicZhang/weather-fe)

- Vue3
- Vite4
- Vue Router
- Vuex
- VueUse
- Element-plus
- Echarts5
- Axios
- Typescript

### [后端](https://github.com/LaicZhang/weather-be)
- Koa2
- Python3

### 数据库
- MongoDB

### 包管理器
- pnpm

### 代码规范
- ESlint
- StyleLint
- ESLint config by antfu
- husky

## 部署
- Vercel
- OSS
- Tencent Cloud
- Tencent Webify
